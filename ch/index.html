部分研究项目介绍

Energy Efficiency Enhancement for CNN-based Deep Mobile Sensing
用于深度移动感知的关键能耗优化方法

目前，深度学习常被用于解决移动感知问题。在大部分解决方案里，为了获得快速实时的响应，深度学习的推理运行在移动设备上，而不是云系统中。然而，移动设备通常都是计算资源和能源受限的平台。此外，一个预测任务通常需要数百亿次数学运算和数千万次参数读取。因此，降低深度学习推理算法的能耗是个关键问题。在这篇小文中，我们简述了三类方法：模型压缩、数据传输优化（专门硬件设计）、以及多种计算资源协同处理，并且针对模型压缩进行了性能评价。

Ruitao Xie, Xiaohua Jia etc.
WCM 2019
Data Transfer Scheduling for Maximizing Throughput of Big-Data Computing in Cloud Systems
大数据计算中的数据传输调度

一个大数据计算任务通常包含三个阶段：计算节点从存储节点读取数据、处理数据、以及将运算结果存回存储节点。因此，存储节点与计算节点之间的数据传输效率会对任务完成时间产生很大的影响。我们提出了一种数据调度方法来降低数据获取时间。该方法将数据获取均衡地分布在存储节点中的多个数据拷贝上，并将传输均衡地分散在存储节点和计算节点之间的多条传输路径上，从而实现了均衡调度。该方法能够避免潜在的拥塞，并且降低数据获取时间。

Ruitao Xie, Xiaohua Jia
TCC 2015
Supporting Seamless Virtual Machine Migration via Named Data Networking in Cloud Data Center
应用于云数据中心的基于数据命名网络的无缝虚拟机迁移方法

云计算以提供弹性服务著称，即计算资源按需分配。这种弹性服务的基础是虚拟化技术，即物理机的资源被分割成若干份，分别创建虚拟机，并且每个虚拟机的资源数量受到动态调节。一个关键性问题是如何在虚拟机迁移（从一个物理机到另一个物理机）的过程中保持服务的可访问性。我们提出了一种能避免服务中断的迁移方法。该方法利用命名网络技术，按照虚拟机提供的服务，对其进行命名，使得服务请求可以由服务名称实现路由，而不再依赖于绑定物理机的IP地址。

Ruitao Xie, Yonggang Wen, Xiaohua Jia, Haiyong Xie
TPDS 2015
Transmission Efficient Clustering Method for Wireless Sensor Networks using Compressive Sensing
无线传感器网络中基于压缩感知的支持高效数据传输的分簇方法

能耗一直是无线传感器网络的一个重要挑战，因为传感器只能携带有限的电池，并且在大多数应用场景中很难及时充电。我们提出了一种降低无线传感节点能耗的数据收集方法。该方法利用压缩感知技术（一种编码方法）来压缩数据，并建立一种最优的簇结构来传输数据。这种簇结构能使数据传输所需的传输次数达到最小，能降低高达60%的能耗。

Ruitao Xie, Xiaohua Jia
TPDS 2014
